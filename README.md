### Project Overview
---
The objective of this project is to help a pizza business understand:
- Revenue performance
- Order behavior across days and months
- Product-level profitability
- Category and size preferences
- Top and bottom selling pizzas
- Operational efficiency trends

This analysis equips decision-makers with the insights needed for menu optimization, inventory planning, pricing strategy, and sales forecasting.

### Tools
---
- SQL Server - For data cleaning, exploration, modeling
- Power BI - For Data visualization, DAX calculation, dashboard design
- DAX - For measures & business metrics

### Key Performance Indicators
---
- Total Revenue
- Total Pizzas Sold
- Total Orders
- Average Order Value
- Daily Trend (Orders by Day of Week)
- Monthly Trend (Orders by Month)
- Revenue by Category
- Revenue by Pizza Size
- Top & Bottom 5 Pizza Products
- Category-level product rankings

### Project Insights
---
Key insights generated from the dashboard include:
- Classic and Supreme pizzas generate the highest revenue.
- Large size pizzas dominate both sales volume and revenue.
- Fridays and weekends drive peak order activity, signaling strong weekend demand.
- Monthly orders show consistent growth, with certain months outperforming others.
- Several pizzas consistently appear in the bottom 5 across revenue, quantity, and orders, indicating candidates for menu restructuring.

These insights help guide strategic decisions in menu optimization, promotional campaigns, inventory forecasting, and operational planning.
